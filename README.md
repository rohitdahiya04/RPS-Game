# Rock Paper Scissors Game

A simple command-line Rock Paper Scissors game written in Python.

## Features

* Play against the computer.
* Random computer choices using Python's `random` module.
* Supports multiple rounds.
* Displays the winner after each round.
* Option to play again or quit.

## Requirements

* Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/rock-paper-scissors.git
```

2. Navigate to the project directory:

```bash
cd rock-paper-scissors
```

3. Run the program:

```bash
python main.py
```

## How to Play

1. Enter one of the following choices:

   * rock
   * paper
   * scissors

2. The computer will randomly select its choice.

3. The winner is determined using the standard rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock

4. After each round, choose:

   * `Y` to play again
   * `N` to quit the game

## Example

```text
Enter your choice (rock, paper, scissors): rock

Player: rock
Computer: scissors

Player WON

Play again (Y/N): y
```
